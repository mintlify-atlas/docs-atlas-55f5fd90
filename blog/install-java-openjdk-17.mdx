---
title: Install Java OpenJDK 17
description: Learn how to install Java OpenJDK 17 on Ubuntu.
---

<Steps>

### Update apt package index

```bash
sudo apt update
```

### Install OpenJDK 17 and JRE

```bash
sudo apt-get install openjdk-17-jdk openjdk-17-jre -y
```

### Verify the installation

```bash
java -version
```

</Steps>
