## Install SSH on Linux

Run the following commands to install the OpenSSH client and server, then enable and start the SSH service:

```bash
sudo apt-get update
sudo apt-get install -y openssh-client openssh-server
sudo systemctl enable ssh
sudo systemctl start ssh

# Either command works to check the status
sudo systemctl status ssh
sudo systemctl status sshd
```

<Note>
  All SSH-related files are stored under `/home/<username>/.ssh` on your system.
</Note>
