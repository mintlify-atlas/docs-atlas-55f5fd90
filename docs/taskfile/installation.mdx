---

## Installation Methods

<Tabs>
  <Tab title="Snap">
    ```bash
    sudo snap install task --classic
    ```
  </Tab>
  <Tab title="Binary">
    Download the binary from the [release page](https://github.com/go-task/task/releases) and add it to your `PATH`.

    ```bash
    wget https://github.com/go-task/task/releases/download/v3.43.3/task_linux_amd64.deb
    sudo dpkg -i task_linux_amd64.deb
    ```
  </Tab>
  <Tab title="Install Script">
    Use the `-b` flag to override the installation directory.

    ```bash
    sh -c "$(curl --location https://taskfile.dev/install.sh)" -- -d -b /usr/local/bin
    ```
  </Tab>
  <Tab title="GitHub Action">
    Install Task in your CI pipeline using the [arduino/setup-task](https://github.com/arduino/setup-task) action.

    ```yaml
    - name: Install Task
      uses: arduino/setup-task@v2
      with:
        version: 3.x
        repo-token: ${{ secrets.GITHUB_TOKEN }}
    ```
  </Tab>
</Tabs>
